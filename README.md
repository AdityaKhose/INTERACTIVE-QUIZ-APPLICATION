#TASK DESCRIPTION#: The Interactive Quiz Application is a front-end web project designed to create an engaging platform for users to test their knowledge through multiple-choice questions. The application displays questions one at a time, allows users to select answers, and calculates the final score. It is built using HTML for structure, CSS for styling, and JavaScript for interactivity. The project focuses on improving front-end development skills such as DOM manipulation, event handling, and responsive design while delivering a simple and user-friendly quiz experience.
